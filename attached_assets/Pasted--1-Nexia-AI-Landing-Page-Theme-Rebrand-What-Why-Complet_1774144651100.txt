#1 - Nexia AI — Landing Page & Theme Rebrand
What & Why
Complete rebrand from "FoxBot" to "Nexia AI". The current landing page has a friendly/childish look that needs to become premium, serious, and exclusive — inspired by versu.ai/cl. This includes a full landing page redesign, new color scheme, new pricing structure ($49.990/$99.990/$199.990 CLP/mes), and switching from a self-serve "Comenzar Gratis" model to a demo/consultation "Agendar Demo" model.

Done looks like
Landing page has a premium, dark, clean design inspired by versu.ai — professional typography, elegant spacing, impact metrics, testimonials section, integration logos
Brand name reads "Nexia AI" everywhere on the landing page and public pages (Register, Login, Demo, Privacy, Terms)
Hero section says something like "IA que vende y atiende por ti" with "Agendar Demo" as the primary CTA (links to a demo scheduling section/form) and "Probar Agente" as secondary
Pricing section shows 3 plans: Starter ($49.990/mes), Pro ($99.990/mes), Enterprise ($199.990/mes) — no free plan visible
Color scheme shifts from green/orange to a premium dark palette with a refined accent color (deep blue or teal accent on dark background)
CSS variables in index.css updated for the new scheme
New logo generated (clean, minimal, no cartoon mascot) and applied as favicon + header
OG meta tags and page titles updated to "Nexia AI"
A "Agendar Demo" section at the bottom with a contact/scheduling form (name, email, company, website, message) that sends to the existing help request system or email
Impact metrics section similar to Versu (e.g., "70% más conversión", "86% menos carga operacional", etc.)
Registration page still works but is de-emphasized (not the main CTA)
Out of scope
Dashboard internal rebrand (separate task)
Admin panel rebrand (separate task)
Widget/chatbot rebrand (separate task)
Backend API changes
Actual calendar/scheduling integration (form submission is enough)
Changing the domain name
